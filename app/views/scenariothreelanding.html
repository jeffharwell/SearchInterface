HIT Instructions

You are going to be asked to write a search that you think would help you answer a specific question. You will be using a new type of search engine that constructs a search diagram using the terms that you type in. You enter your query into the three boxes at the top of the screen. You will be typing a sentence but you put the different parts of the sentence into different boxes (subject, predicate and object if that distinction help you). You don't need to put something in each box.

For example:

In the below example a user is trying to find an answer to the question "Why won't my car start!".

The user first typed "my car" in the first box, "won't" in the second box, and "start" in the third box and then clicked the "Add Terms!" button to add the terms to the search diagram.

Thinking that perhaps this search would be too generic the user is getting ready to add the terms "the keys" (in the first box), "won't" (in the second box) "turn" (in the third box). When the "Add Terms!" button is clicked the computer will add those terms to the search diagram.

The "Start Over" button removes all search terms from the graph so that you can start fresh

The "Search!" button will perform the search. You only need to do one search so when you click this button you will be presented with results and you won't be able to make further modifcations to your search.

Just do you best with the search, there is no right answer!

Once you have submitted your search scroll to the bottom of the results list and click the red button labeled "Take the Survey!" to proceed to the survey. Once you have finished the survey you will be taken back to Amazon Mechanical Turk and your work will automatically be submitted.
